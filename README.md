# HayaKoh's Station Clicksign
MinecraftのClickSignsMod用の駅看板を追加する拡張リソースパック

## ⚠注意 Attention⚠
This resource pack has been created for Japanese users, so it may be difficult to use in other languages.
Please only use this resource pack if you can accept this limitation.<br>
このリソースパックは日本語使用者向けに作成されているため、他の言語において使用が困難な場合があります。
この点にご理解頂ける場合のみこのリソースパックをご利用ください。

Please note that all of the following text is also written in Japanese.<br>
以下の文章もすべて日本語で書かれています。

## 前提
前提ModとしてClick Signs（Clickism氏による）が必要です。
以下からダウンロードしてください。

（modrinth）https://modrinth.com/mod/clicksigns

## 特徴
日本国内の駅名標をMinecraft内で再現することを目的に作成しました。
現在は以下の1種類（矢印差分などを除く）が利用できます。

- JR西日本（矢印、路線色の差分あり）

ただし、文字の記入後に別種の駅名標に切り替えた場合は表示崩れが生じる場合があります。
